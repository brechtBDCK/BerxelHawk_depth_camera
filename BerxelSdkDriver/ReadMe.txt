Example Program Instructions:
1. Install Python's pip tool

2. Install numpy
  Command: pip install numpy

3. Install opencv-python for drawing/visualization
  Command: pip install opencv-python


SDK Usage Instructions:
1. The SDK uses the DLL libraries located in Berxel\BerxelSDK\lib\x64.
2. You can modify the library path as needed for your project by editing the Berxel\BerxelSDK\Python\BerxelSdkDriver\BerxelHawkNativeMethods.py file.
